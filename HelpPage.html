Shape Editor Help

Choose Shape

Select a shape from the dropdown to have appear when drawing on the canvas.

Drawing Modes

Select a drawing mode either: draw, translate, scale, rotate or select.

Every mode except draw can be used by clicking on the desired shape to transform.

Select mode is used for selecting object to COPY and PASTE.

Shape Thickness and Color

Select a shape thickness from a dropdown list. Pick a a color for the shape to be drawn with. The default color is white.

Drawing Shapes

Select a shape from the dropdown and click and drag on the canvas to draw it for rubber banding shapes.

WARNING: curve and polyline need to be drawn with 4 clicks for their points.

Rubber band shapes

This includes shapes: Line, Triangle, Square, Rectangle, Circle, Ellipse, and Polygon

For these shapes a "ghost" shape will draw to give you an example of how the shape will look while the mouse is down.

Once you lift up on the mouse the shape will draw in the same way as the last seen "ghost" shape.

Multi-point shapes

This includes shapes: Polyline and Curves

For these shapes no "ghost" shape will draw. Once you click 4 times the shape will appear according to the users clicks on the canvas.

Undo

Click the Undo button to undo your last change. This can also be done with ctrl + the z key.

Save and Load

Save your diagram as a JSON file and load it later to bring the shapes back onto the canvas.

WARNING loading a JSON file will get rid of any shapes on the canvas. So save anything you want to keep!

Save as JPEG / Export to PDF

Use the Save as JPEG or Export to PDF buttons to download your work as an image or a PDF and also name the file.

Copy and Paste

Select a shape, press the copy button or Ctrl+C to copy it, and press the past button or Ctrl+V to paste a duplicated shape.

Grid and Snap

Toggle the grid visibility and snapping with the toggle grid button to make precision drawing easier.

WARNING this grid will show up in any saved PDFs or JPEGs if it isn't toggled off with the toggle button.
